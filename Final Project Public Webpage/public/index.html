Welcome!

This website acts as the final project for ECE 413/513 at the University of Arizona. It allows a user to create an account, register one or multiple HeartTracker device(s), and view the output of a given registered device for either the last week or a specific day.

Get Started!

To start looking through the final project, go to the navigation bar at the top of the webpage and select "Sign Up"; you will be prompted to make an account, including the ID of your Arduino, so make sure you have that ready. If you already have an account, you can click on "Log In" to move straight to your main page. If you want to see any third-party resources used in the making of this project, click on "References" to see a full list!"
